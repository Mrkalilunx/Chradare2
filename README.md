# Chradare 安装指南

[前往官方原版 Radare2 项目](https://github.com/radareorg/radare2)

## 安装方法

### 从源码安装

```bash
git clone https://github.com/Mrkalilunx/Chradare2
cd Chradare2
bash sys/install.sh
```

## 验证安装

安装完成后，运行以下命令验证：

```bash
r2 -v
```

## 开始使用

```bash
r2 /path/to/binary
```

更多使用方法和详细文档请参考项目内的说明文件。

---

Chradare 是基于 Radare2 的汉化版本，保留所有原版功能并提供中文支持。
